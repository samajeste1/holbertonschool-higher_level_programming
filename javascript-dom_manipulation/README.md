# JavaScript DOM Manipulation

This project covers JavaScript DOM manipulation techniques including:

- Selecting HTML elements using `querySelector`, `getElementById`, and class selectors
- Modifying element styles and content
- Adding/removing CSS classes
- Creating and appending new DOM elements
- Making HTTP requests using the Fetch API
- Listening to DOM and user events
